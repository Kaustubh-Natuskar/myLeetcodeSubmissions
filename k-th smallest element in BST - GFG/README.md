# k-th smallest element in BST
## Medium 
<div class="problem-statement">
                <p></p><p><span style="font-size:18px">Given a BST and an integer K. Find the Kth Smallest element in the BST.&nbsp;</span></p>

<p><span style="font-size:18px"><strong>Example 1:</strong></span></p>

<pre><span style="font-size:18px"><strong>Input:
&nbsp;     </strong>2
&nbsp;   /   \
&nbsp;  1     3<strong>
</strong>K = 2
<strong>Output: </strong>2</span>
</pre>

<p><span style="font-size:18px"><strong>Example 2:</strong></span></p>

<pre><span style="font-size:18px"><strong>Input:
</strong>        2
&nbsp;     /  \
&nbsp;    1    3
K = 5
<strong>Output: </strong>-1</span></pre>

<p>&nbsp;</p>

<p><span style="font-size:18px"><strong>Your Task:</strong><br>
You don't need to read input or print anything. Your task is to complete the function&nbsp;<strong>KthSmallestElement()&nbsp;</strong>which takes the root of the BST and integer K as inputs and return the Kth smallest element in the BST, if no such element exists return -1.</span></p>

<p><span style="font-size:18px"><strong>Expected Time Complexity:&nbsp;</strong>O(N).<br>
<strong>Expected Auxiliary Space:&nbsp;</strong>O(1).</span><br>
<br>
<span style="font-size:18px"><strong>Constraints:</strong><br>
1&lt;=Number of nodes&lt;=100000</span></p>
